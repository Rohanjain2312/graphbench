## Metrics

| Metric | Description |
|--------|-------------|
| Exact Match (EM) | 1.0 if normalised prediction == normalised gold |
| Token F1 | Token-level precision/recall F1 |
| Latency p50 | Median wall-clock time per question (ms) |
| Latency p95 | 95th percentile latency (ms) |
